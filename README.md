# Luxury SaaS Website

This is a Next.js 14 project showcasing a premium, interactive, and modern website design. It emphasizes luxury aesthetic, modern UI/UX, and cinematic animations ideal for a SaaS or creative studio site.

## Features

- Modern dark theme with glassmorphism
- Premium typography and smooth animations
- Highly responsive design

## Getting Started

Install the dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Start the production build:

```bash
npm start
```
